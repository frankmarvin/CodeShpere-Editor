# 🧠 AI Safety & Data Handling Policy

## AI Integration Principles

- User prompts are not stored without consent
- No training on user code
- API keys stored securely
- Transparent AI responses

---

## Data Handling

If AI APIs are used:
- Requests are encrypted (HTTPS)
- No long-term retention unless stated
- Logs anonymized when possible

---

## Responsible AI Usage

- AI should not generate malicious code
- Unsafe or harmful outputs are filtered
- Users remain responsible for reviewing AI-generated code
